# Decodelabs Cyber Security Internship Projects

## Overview

This repository contains the projects completed during the **Decodelabs Cyber Security Internship Program**. The projects cover various cybersecurity domains including password security, encryption, phishing awareness, and system security auditing.

The objective of these projects is to develop practical cybersecurity skills and gain hands-on experience with common security concepts, risk assessment techniques, and defensive security practices.

---

# Projects Included

## 1. Password Strength Checker

### Objective

Develop a Python-based tool to analyze password strength and help users create secure passwords.

### Features

* Checks password length
* Detects uppercase and lowercase characters
* Detects numbers and special symbols
* Provides password strength ratings
* Gives recommendations for improvement

### Technologies Used

* Python 3

---

## 2. Basic Encryption and Decryption Tool

### Objective

Create a simple encryption and decryption utility for secure text communication.

### Features

* Encrypt plain text messages
* Decrypt encrypted messages
* User-friendly command-line interface
* Demonstrates basic cryptography concepts

### Technologies Used

* Python 3

---

## 3. Phishing Awareness and Detection Project

### Objective

Educate users about phishing attacks and help identify suspicious websites, emails, and social engineering attempts.

### Features

* Explains common phishing techniques
* Demonstrates phishing indicators
* Provides cybersecurity awareness guidelines
* Helps users recognize malicious links and emails

### Technologies Used

* Python
* Research and Documentation

---

## 4. Audited Systems – A Framework for Blue Team Defense

### Objective

Perform a basic system security audit using Blue Team methodologies and evaluate the overall security posture of a computer system.

### Features

* Identity Security Assessment
* Password and MFA Review
* Software and Patch Management Analysis
* User Account and Privilege Review
* Firewall and Antivirus Assessment
* Risk Assessment and Security Recommendations

### Technologies Used

* Microsoft Windows
* Windows Security
* Microsoft Defender
* Security Audit Documentation
