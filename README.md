# Decodelabs Internship Projects

## Overview

This repository contains the projects completed during the Decodelabs Cyber Security Internship Program. The projects focus on cybersecurity awareness, password security, encryption techniques, and phishing attack prevention.

## Projects Included

### 1. Password Strength Checker

**Objective:**
Develop a Python-based tool to analyze password strength and help users create secure passwords.

**Features:**

* Checks password length
* Detects uppercase and lowercase characters
* Detects numbers and special symbols
* Provides password strength rating
* Gives recommendations for improvement

**Technologies Used:**

* Python 3

---

### 2. Basic Encryption and Decryption Tool

**Objective:**
Create a simple encryption and decryption utility for secure text communication.

**Features:**

* Encrypt plain text messages
* Decrypt encrypted messages
* User-friendly command-line interface
* Demonstrates basic cryptography concepts

**Technologies Used:**

* Python 3

---

### 3. Phishing Awareness and Detection Project

**Objective:**
Educate users about phishing attacks and help identify suspicious websites and emails.

**Features:**

* Explains common phishing techniques
* Demonstrates phishing indicators
* Provides cybersecurity awareness guidelines
* Helps users recognize malicious links and emails

**Technologies Used:**

* Python
* Documentation and Research
