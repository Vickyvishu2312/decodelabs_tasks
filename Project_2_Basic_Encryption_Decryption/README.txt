# Basic Encryption and Decryption

## Project 2 - Cyber Security

### Overview

This project demonstrates a simple encryption and decryption system using the Caesar Cipher technique. The program converts normal text into encrypted text and then decrypts it back to its original form.

### Objective

The objective of this project is to understand the basic concept of cryptography and learn how information can be protected using simple encryption methods.

### Features

* Encrypts user-entered text
* Decrypts encrypted text back to original text
* Supports uppercase and lowercase letters
* Allows the user to enter a custom shift value
* Preserves spaces and special characters

### Technologies Used

* Python 3
* Visual Studio Code

### Working

1. User enters a message.
2. User enters a shift value.
3. The program encrypts the message by shifting letters.
4. The encrypted message is displayed.
5. The program decrypts the encrypted message.
6. The original message is displayed again.

### Concepts Used

* Strings
* Loops
* Conditional Statements
* ASCII Values
* ord() Function
* chr() Function
* Caesar Cipher Technique
