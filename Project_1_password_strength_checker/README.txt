# Password Strength Checker

## Project 1 - Cyber Security

### Overview

This project is a simple Password Strength Checker developed in Python. The program analyzes a password entered by the user and determines whether it is Weak, Medium, or Strong based on different security conditions.

### Objective

The main objective of this project is to understand the basic concepts of password security and implement them using Python programming.

### Features

* Checks password length
* Detects uppercase letters
* Detects numeric digits
* Detects special symbols
* Classifies password strength as:

  * Weak
  * Medium
  * Strong

### Technologies Used

* Python 3
* Visual Studio Code

### Working

1. User enters a password.
2. The program checks:

   * Length of the password
   * Presence of uppercase letters
   * Presence of numbers
   * Presence of special characters
3. Based on the conditions met, the password strength is displayed.
