```
# Audited Systems – A Framework for Blue Team Defense
```

```
## Overview
```

```
This project was completed as part of the Decodelabs Cyber Security Internship
Program (Week 4 Project).
```

```
The objective of this project is to perform a basic security audit of a computer
system using a Blue Team approach. The assessment focuses on identifying
potential vulnerabilities, reviewing existing security controls, and
recommending security improvements.
```

```
---
```

```
## Project Objectives
```

```
 Understand Blue Team defensive security concepts.
 Evaluate system security posture.
 Review password and authentication practices.
 Assess software update and patch management status.
 Examine user security awareness and access controls.
 Verify network and endpoint protection measures.
 Generate a security audit report with recommendations.
```

```
## Key Recommendations
```

`1. Enable Multi-Factor Authentication (MFA).` 

`2. Use strong and unique passwords.` 

`3. Keep Windows and applications updated.` 

`4. Maintain active antivirus protection.` 

`5. Keep firewall protection enabled.` 

`6. Remove unused software and user accounts.` 

`7. Regularly back up important data.` 

`8. Conduct periodic security assessments.`
